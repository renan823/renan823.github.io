Hotel 5 estrelas - Classe A

A melhor estadia oferecida por aqui... Não é para qualquer um! O hotel "Luxor class A" possui uma ampla gama de atividades, tanto para os aventureiros quanto para os que buscam relaxar. Na órbita baixa da terra (LEO), apresenta um conforto inigualável, e consegue hospedar até duas famílias.

Hotel 4 estrelas - Classe B

Uma opção para quem busca as melhores atividades radicais, incluindo uma piscina de borda finita e um bungee jump espacial. Suas acomodações comportam até uma família, hospedada com o maior conforto dentro das 5 salas internas.

Hotel 4 estrelas - Classe B

Uma escolha cômoda para quem não pretende investir tantos tostões. A cápsula é integrada a módulos de carga, que permitem um abastecimento constante para a manutenção da comodidade. Sua estrutura contém uma sauna solar e quartos para 3 pessoas.

Hotel 3 estrelas - classe C

A escolha mais econômica no mercado! O “Hotel class C” acomoda apenas 2 pessoas, um passeio para casais em, literalmente, lua de mel. A estrutura interna é ampla, e conta com um pequeno spa integrado (não inclui massagistas), juntamente com uma suíte.

Passeio turístico - atmosfera

Este é o ÚNICO passeio espacial "all inclusive" com preços acessíveis no mercado. O tour conta com guias especializados (astrônomos) para uma melhor apreciação da viagem. A nave comporta 5 pessoas + 2 guias simultaneamente.
